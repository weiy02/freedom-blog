## 最新文章

> 在 `docs` 目录下按分类创建文章，并为每篇文章配置正确的 frontmatter：
>
> ```yaml
> ---
> title: 文章标题
> date: 2026-01-01
> categories:
>   - 前端
>   - JavaScript
> tags:
>   - Vue
>   - VuePress
> ---
> ```

分类、标签、归档页会在构建时根据 frontmatter **自动生成**，你可以通过顶部导航栏进入：

- **时间轴**：`/timeline/`
- **分类**：`/categories/`
- **标签**：`/tags/`
