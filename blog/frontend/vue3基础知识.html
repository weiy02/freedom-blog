箭头函数 1.箭头函数没有独立的 this、arguments 和 super 绑定，并且不可被用作方法。 2.箭头函数不能用作构造函数。使用 new 调用它们会引发 TypeError。它们也无法访问 new.target 关键字。 3.箭头函数不能在其主体中使用 yield，也不能作为生成器函数创建。

x => x*x
// 相当于
function (x) {
	return x*x;
}
